int setupSHM_GameStart ( struct Share *  gs)

Takes a struct and inserts it into a SHM Segment.

This function is customized to fit the Game Start Data (see Milestone 0). It creates (createSHM) a SHM Segment. Attaches the Segment to the ptrGameStart adress and the fills it with the content of gs. the nested players struct is constructed with a while loop because it must be able to handel a variable amount of players (see Milestone 2).
